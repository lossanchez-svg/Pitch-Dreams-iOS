## How to generate

1. Open https://aistudio.google.com/prompts/new_chat?model=gemini-3-pro-image-preview
2. Make sure your paid API key is linked (it should still be from prior session)
3. For each animal: start a NEW chat, paste stage 1 prompt, run, then in the SAME chat paste stage 2 (this preserves character identity), then stage 3
4. Right-click each generated image → Save As → use the exact filename listed above
5. Save all into `/Users/lossa/Pitch-Dreams-iOS/ArtAssets/Avatars/`

## Master constraints (apply to every prompt)
- Cel-shaded anime/manga art style (Inazuma Eleven / Blue Lock / Beastars)
- Bold black outlines, vibrant saturated flat colors, two-tone shading
- Single character, centered, full body visible
- Plain solid white background, square 1:1 aspect ratio
- ANTHROPOMORPHIC ANIMAL — NOT a human, NOT a person in a costume
- Stage 1 = chibi cub (3-4 heads tall, oversized jersey)
- Stage 2 = lean teen (5 heads, fitted jersey, dynamic action pose)
- Stage 3 = powerful adult (premium jersey with gold/silver trim, captain armband, glowing eyes, energy effects)
